# My Flask K8s Service

Простой веб-сервис на Flask, упакованный в Docker и развернутый в Kubernetes (Minikube) и Яндекс Облаке.

## 📋 О проекте

Сервис предоставляет REST API с информацией о статусе, здоровье и версии приложения. Проект демонстрирует полный цикл разработки и развертывания: от кода до облака.

## 🛠 Технологии

| Технология | Назначение |
|------------|------------|
| Python 3.9 + Flask | Веб-сервис |
| Docker | Контейнеризация |
| Docker Hub | Реестр образов |
| Kubernetes (Minikube) | Оркестрация контейнеров |
| Yandex Cloud | Облачное развертывание |
| Git + GitHub | Контроль версий |

## 📡 API Endpoints

| Endpoint | Метод | Описание | Пример ответа |
|----------|-------|----------|---------------|
| `/` | GET | Главная страница | `{"service": "Flask K8s Demo", "status": "running", "timestamp": "2026-06-07T12:00:00", "hostname": "pod-name"}` |
| `/health` | GET | Проверка здоровья | `{"status": "healthy"}` |
| `/info` | GET | Информация о сервисе | `{"version": "1.0.0", "python_version": "3.9", "framework": "Flask"}` |
